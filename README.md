## hello-rails-react

> Display the random greeting on a button click

## Setup
>- Git clone `https://github.com/ramrod433/rails-react.git`
>- cd rails-react
>- bundle install
>- rails db:create db:migrate
>- npm install
>- npm run build
>- rails server

## Built With
>- Webpack / React
>- Rails 7.0
